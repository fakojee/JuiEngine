Release Notes

JuiEngine 1.0

(08/04/2025)
!Not a Public Release!

  • Gender Selection Added

    You can set the character's gender as Female, Male, or Unspecified.

  • Race Selection Added

    You can set the character's race as Caucasian, Asian, Latino, or Unspecified.

  • World Type Selection Added

    You can set the character's world as Real World, Futuristic, Medieval, or Sci-Fi.

  • NSFW Added

    You can adjust the intensity of NSFW content via a slider after enabling it.

  • Story Mode Added

    When active, Story Mode writes the scenario, character personality, and greeting message.

  • Topics Added

    90 + 23 (NSFW) topics were added. Topics can affect many aspects of the story, including its direction and scenario.
